13. SQL vs NoSQL

Diferencias fundamentales en el acceso.

Relacionales (SQL): Normalizan datos en tablas (filas/columnas) vinculadas mediante identificadores (Primary/Foreign Keys).

NoSQL: Diseñadas para alta escalabilidad y baja latencia. Ideales para datos semiestructurados o análisis masivos sin esquemas fijos.

Característica Relacional NoSQL
Estructura Tablas rígidas Documentos/Grafos
Relación Identificadores Embebidos/Sin ID
