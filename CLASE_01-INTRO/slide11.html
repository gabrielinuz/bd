11. El SGBD / DBMS

Sistema Gestor de Base de Datos.

El Data Base Management System (DBMS) es el software que actúa como capa de abstracción entre el usuario y los datos físicos.

Funciones núcleo:

  • Creación: Definición de esquemas y estructuras.
  • Gestión: Administración de permisos y seguridad.
  • Eficiencia: Optimización de búsquedas y almacenamiento.

Clasificación Actual:

  • Relacionales (SQL)
  • No Relacionales (NoSQL)
