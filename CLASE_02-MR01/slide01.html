01. El Modelo Relacional

RDBMS: Relational Database Management System.

Basado en la teoría de conjuntos, el modelo relacional organiza los datos en Tablas (Relaciones).

  • Tuplas: Filas que representan registros únicos.
  • Atributos: Columnas que definen propiedades del dato.
  • Dominios: El conjunto de valores permitidos para cada atributo.

Dato Técnico: Un RDBMS es el software (Motor) que implementa este modelo matemático para garantizar la integridad y persistencia.
