18. Ventajas de los Esquemas

Seguridad y modularización en PostgreSQL.

Permite que varios usuarios usen la misma base de datos sin colisiones de nombres.

    -- Crear esquema y tabla
    CREATE SCHEMA rrhh;
    CREATE TABLE rrhh.empleados (...);
    
    -- Acceso calificado
    SELECT * FROM rrhh.empleados;

Uso en Clase: Ideal para separar entornos de producción, auditoría y pruebas dentro del mismo motor.
