11. Cláusulas de Consulta

Filtrado y ordenamiento eficiente.

Optimizan la recuperación de datos para no sobrecargar la red (GREENSOFT):

    SELECT nombre, edad 
    FROM Users 
    WHERE edad > 18 
    ORDER BY nombre ASC 
    LIMIT 10;
  • WHERE: Filtro lógico.
  • ORDER BY: Organización visual.
  • LIMIT: Paginación (vital para el rendimiento).
