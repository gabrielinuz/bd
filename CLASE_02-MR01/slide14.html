14. Agrupamiento Lógico

GROUP BY y el filtro post-agregación HAVING.

GROUP BY: Colapsa filas con valores idénticos en grupos para aplicar funciones de agregado.

HAVING: Es el WHERE de los grupos. Se usa cuando el filtro depende de una función de agregado.

    SELECT pais, COUNT(*) 
    FROM Usuarios 
    GROUP BY pais 
    HAVING COUNT(*) > 100;

Rigor Técnico: El orden de ejecución es: FROM -> WHERE -> GROUP BY -> HAVING -> SELECT.
