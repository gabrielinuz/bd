04. Implementación de Índices

Sintaxis estándar para los tres motores (MySQL, SQLite, PostgreSQL):

    -- Crear índice básico
    CREATE INDEX idx_usuarios_email ON Users(email);
    
    -- Crear índice único (evita duplicados)
    CREATE UNIQUE INDEX idx_dni ON Personas(dni);

Nota: En PostgreSQL, los índices se crean automáticamente para PK y UNIQUE constraints.
